# Better HN: Realtime Comment Updates & Cleaner Look

For those who read HN comments section a lot.

1. It updates comments without the page reload — instantly see new ones as they appear!

2. Cleaner comment topline for less distraction — shows controls only on topline hover

![Image](https://github.com/user-attachments/assets/4b98835f-80cb-4656-abdd-fd802d04c913)

# Install

1. Get Tampermonkey (<a href="https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en&pli=1">Chrome</a>, <a href="https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/">Firefox</a>)

2. Add the userscript https://github.com/xpl/better-hn/blob/main/userscript.js
